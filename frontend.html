FRONTEND

El Frontend se refiere a la parte de un sitio web o aplicación con la que los usuarios interactúan directamente. Es la interfaz gráfica de usuario que permite la navegación, interacción y uso del contenido de una página o aplicación. Los desarrolladores frontend se encargan de crear y optimizar la interfaz visual para que sea atractiva, accesible y funcional en diferentes dispositivos, navegadores y resoluciones.

Fundamentos de frontend

  • HTML: Lenguaje de marcado para estructurar contenido web.
  • CSS: Lenguaje para definir el diseño y estilo de las páginas web.
  • JavaScript: Lenguaje de programación para agregar interactividad.
  • Frameworks de JavaScript: React, Angular, Vue.js para mejorar interfaces.
  • Responsive Design: Técnicas para que el diseño se adapte a todos los dispositivos.
  • Versionamiento: Git para gestionar los cambios del código.
  • Accesibilidad Web: Buenas prácticas para personas con discapacidades.
  • Optimización de Rendimiento: Técnicas para mejorar el tiempo de carga.
  • Preprocesadores CSS: Sass y LESS para escribir CSS de manera eficiente.
  • SEO: Optimización para mejorar la visibilidad en motores de búsqueda.
